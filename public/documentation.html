ParaFlix Developer Instructions

ParaFlix is a web application that allows paranormal enthusiasts to access a variety of information about paranormal movies. Users can register an account, update their personal information, and create a list of their favorite movies. ParaFlix consists of a well-designed REST API and architected database built using JavaScript, Node.js, Express, and MongoDB. The REST API is accessed via commonly used HTTP methods like GET and POST. The CRUD method is used to retrieve data from the database and store that information in a non-relational way.

Application Features

  • Returns a list of ALL movies to the user
  • Returns data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Returns data about a genre (description) by name/title (e.g., “Thriller”)
  • Returns data about a director (bio, birth year, death year) by name
  • Allows new users to register
  • Allows user to update their user info (username)
  • Allows user to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later)
  • Allows user to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later)
  • Allows existing users to deregister (showing only a text that a user email has been removed—more on this later)
